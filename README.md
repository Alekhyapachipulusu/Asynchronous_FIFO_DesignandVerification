This project is the design and verification of Asynchronous FIFO

Motivation:In Real time systems not all components and pherpherials operate on the same freuency therefore while moving data from one system to another we need an asynchrnous buffer that can efficiently perform write and read operations preventing
            metastability and raise the empty and full flags without causing data corruption.

Learning:Through this project I learnt and reserached about the different hardware solutions to resolve the metastability issue one of which is the Asynchronous FIFO,
